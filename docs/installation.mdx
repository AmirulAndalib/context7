title: "Installation"
description: "Install and connect the Context7 MCP server in your editor."
---

## Requirements

- Node.js >= 18
- An MCP-capable editor (Cursor, Claude Code, VS Code, Windsurf, etc.)
- Optional: Context7 API key for higher limits and private repos (`https://context7.com/dashboard`)

## Cursor (remote server)

```json
{
  "mcpServers": {
    "context7": {
      "url": "https://mcp.context7.com/mcp",
      "headers": { "CONTEXT7_API_KEY": "YOUR_API_KEY" }
    }
  }
}
```

## Cursor (local server)

```json
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp", "--api-key", "YOUR_API_KEY"]
    }
  }
}
```

For other clients (Claude Code, Zed, VS Code, etc.), see their examples in README.
