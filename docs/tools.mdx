## resolve-library-id

Resolves a general library name to a Context7 ID.

- input: `libraryName`
- output: Context7-compatible ID (e.g., `/vercel/next.js`)

## get-library-docs

Fetches docs/snippets for a library.

- input: `context7CompatibleLibraryID` (required), `topic` (optional), `tokens` (optional)
- behavior: returns up-to-date code examples and docs
