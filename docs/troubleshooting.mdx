## Module not found

Use bunx instead of npx:

```json
{
  "mcpServers": {
    "context7": { "command": "bunx", "args": ["-y", "@upstash/context7-mcp"] }
  }
}
```

## ESM resolution issues

```json
{
  "mcpServers": {
    "context7": { "command": "npx", "args": ["-y", "--node-options=--experimental-vm-modules", "@upstash/context7-mcp@1.0.6"] }
  }
}
```

## TLS/cert issues

```json
{
  "mcpServers": {
    "context7": { "command": "npx", "args": ["-y", "--node-options=--experimental-fetch", "@upstash/context7-mcp"] }
  }
}
```
